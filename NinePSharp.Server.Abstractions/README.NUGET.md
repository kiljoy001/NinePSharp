# NinePSharp.Server.Abstractions

Shared abstractions for building NinePSharp-compatible backends.

## What is included
- Backend contracts (filesystem/protocol interfaces).
- Shared configuration models and utility abstractions.

## Install
```bash
dotnet add package NinePSharp.Server.Abstractions
```

Typically used together with `NinePSharp` and one or more `NinePSharp.Backends.*` packages.
