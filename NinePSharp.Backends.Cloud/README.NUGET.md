# NinePSharp.Backends.Cloud

Cloud backend plugin for storage and secret services via 9P.

## Included integrations
- AWS: S3, Secrets Manager.
- Azure: Blob Storage, Key Vault.
- GCP: Cloud Storage, Secret Manager.

## Install
```bash
dotnet add package NinePSharp.Backends.Cloud
```

Mount cloud resources as filesystem-like paths in your NinePSharp host.
